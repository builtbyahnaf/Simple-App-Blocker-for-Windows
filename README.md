## **Simple App Blocker for Windows**

A fully working bloat-free app blocker for Windows written with pure C++ and...nothing else.


**How to use:**

- Go to Start

- Find the app you want to block

- Right click, go to 'More' and select 'Open file location'. It will redirect you to a shortcut.

- Right click and select 'Open file location'. It will take you to where the launching exe of the app is.

- Now open the blocker app

- Click 'Choose EXE'. It will open a popup dialog. Navigate to where the launching exe is. The best way is to simply copy and paste the path.

- Select that EXE

- Choose for how many days to block

- Click 'Block' and done!




**Features:**

- No background services needed. (Doesn't steal your data)

- No installation required. Just run the exe as administrator.

- You'll have to directly select the launching exe of that specific app.




**Known Issues:**
- UI glitches while resizing
- The block data isn't protected, so there's a risk of manually editing the file to cheat. (Don't do that to stay loyal)
